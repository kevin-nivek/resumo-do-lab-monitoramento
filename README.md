# resumo-do-lab-monitoramento
Este repositorio contem o resumo das lições aprendidas durante o desenvolvimento do lab de ferramentas de Monitoramento de recursos na DIO

* Assistente do Azure
   Analisa os recursos implantadose faz recomendações com base nas práticas recomendadas
  
* Integridade do Serfiço do Azure
  Exibição focada  apenas nos serviços e regiões que se usa

* Azure Monitor
  Traz uma visão "de tudo aquili que tenho dentro de casa" , mas é de suma importancia fazer a validação, trazer as metricas como tambem os logs dentre outros

    * Azure service health
     Validações de serviços
    
    * Advisor 
      centro de recomendações, apresenta ideias para ajustar problemas
